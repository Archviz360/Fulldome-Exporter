# Fulldome Converter Documentation

## Overview

Fulldome Converter is a powerful tool for converting 360-degree videos and photos into fulldome format, designed specifically for planetariums and dome projections.

## Installation

### From Source
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run `python src/fulldome_converter.py`

### Standalone Executable
1. Download the latest release
2. Extract the zip file
3. Run `Fulldome_Converter.exe`
